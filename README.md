## üìò What is RIGEO?

RIGEO is software designed to help organize and improve task scheduling in fog computing environments. It uses advanced methods called metaheuristic algorithms to find the best way to handle tasks. This software runs inside MATLAB and includes 8 different methods, including the new RIGEO approach. 

Fog computing means running computing and data storage close to the devices that use them, instead of all relying on distant data centers. This helps reduce delay and saves energy. RIGEO focuses on making this process smart and efficient.
